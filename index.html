CSS

CSS Selectors

Element Description Example
* Applies CSS properties to all elements of the webpage 

* {
    /* Css properties not specified */
}
h1 Applies CSS properties to all h1 elements of the webpage. Works on all other elements as well 

h1 {
    /* Css properties not specified */
}

Values for every attribute

font-variant

Value Description Example
small-caps Applies Small Caps style to text. 

* {
    font-variant: small-caps;
}

cursor

Value Description Example
default Changes to default cursor when hovered over the element. 

* {
    cursor: default;
}
